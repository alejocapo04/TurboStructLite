# TurboStructLite

TurboStructLite is the free/open-source edition of TurboStruct: an asynchronous persistence system designed to run the save/load pipeline off the Game Thread to avoid gameplay hitches, with crash-safe atomic commits (anti-corruption) and schema evolution support.
